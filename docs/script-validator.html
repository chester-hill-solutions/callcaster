CallCaster Script JSON Validator

Use this tool to validate your CallCaster script JSON files before uploading them to the platform. The validator will check for common errors and provide warnings about potential issues.

Example Script JSON Structure

Here's a minimal example of a valid script JSON structure:

{
  "pages": {
    "page_1": {
      "id": "page_1",
      "title": "Introduction",
      "blocks": ["block_1", "block_2"]
    }
  },
  "blocks": {
    "block_1": {
      "id": "block_1",
      "type": "textarea",
      "title": "Greeting",
      "content": "Hello, this is [Name] from [Company].",
      "options": [],
      "audioFile": ""
    },
    "block_2": {
      "id": "block_2",
      "type": "select",
      "title": "Initial Question",
      "content": "Do you have a few minutes to talk?",
      "options": [
        {
          "content": "Yes",
          "next": "end"
        },
        {
          "content": "No",
          "next": "end"
        }
      ],
      "audioFile": ""
    }
  }
}
